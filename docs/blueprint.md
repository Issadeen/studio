# **App Name**: TruckLogistics Monitor

## Core Features:

- Input and Permit Generation: Input truck number, date, product, and company information. Generate and store permit number upon submission.
- Pre-Check Tracking: Track pre-check status with pre-check checkbox and date. Pre-check expiry countdown stops when loaded.
- Expiry Monitoring: Monitor pre-check expiry (72 hours) and provide visual alerts for expiring pre-checks. Incorporate quizzes to confirm status.

## Style Guidelines:

- Primary color: Neutral grays for a professional look.
- Secondary color: Light blue for interactive elements.
- Accent color: Teal (#008080) to highlight important information and CTAs.
- Responsive layout for accessibility across devices.
- Clear and intuitive form design for easy data input.
- Use clear icons to represent different data fields and actions.
